## 목표
어떤 회사를 가서도 사용할 수 있는 **나만의 QA 웹**

## 웹에 필요한 기능들
1. Senior ( 중요도 : 최상 )
2. Knowledge Base ( 중요도 : 하 )
3. Words Conventions ( 중요도 : 중상 )
4. Company Features ( 중요도 : 최상 )


## 포트폴리오로서 집중해야하는 기능들
1. **개발자**로써 개발부터 playwright 테스트자동화까지 도입해서 ci/cd 까지 연결 후, 배포까지
2. **PM**으로써 Claude 를 이용해 처음 계획부터 끝까지 자동화
3. **QA**로써 AI를 이용해 업무 효율을 향상
